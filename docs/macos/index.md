# macOS

## Keyboard & Mouse

Show all devices connected via USB:

``` console
ioreg -p IOUSB
```

For Logitech devices install [Logi-Options+](
https://www.logitech.com/de-de/software/logi-options-plus) to configure the
device and the special keys.

``` console
brew install --cask logi-options+
```
