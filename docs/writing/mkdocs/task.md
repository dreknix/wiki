* https://squidfunk.github.io/mkdocs-material/reference/lists/#tasklist
* https://facelessuser.github.io/pymdown-extensions/extensions/tasklist/

## Setup

```yaml
markdown_extensions:
  - pymdownx.tasklist:
      custom_checkbox: true
```
